Agent Skills 是指令、脚本和资源的文件夹，代理可以发现并使用它们来更准确、更高效地完成任务。

## 为什么需要 Agent Skills？

代理的能力日益增强，但往往缺乏可靠地完成实际工作所需的上下文。Skills 通过让代理能够按需加载程序性知识以及特定于公司、团队和用户的上下文来解决这个问题。能够访问一组 skills 的代理可以根据他们正在处理的任务扩展其能力。

**对于技能作者**：构建一次能力，即可部署到多个代理产品中。

**对于兼容的代理**：对 skills 的支持让最终用户能够开箱即用地赋予代理新的能力。

**对于团队和企业**：将组织知识捕获到可移植、版本控制的包中。

## Agent Skills 能实现什么？

- **领域专业知识**：将专业知识打包成可复用的指令，从法律审查流程到数据分析管道。
- **新能力**：赋予代理新的能力（例如创建演示文稿、构建 MCP 服务器、分析数据集）。
- **可重复的工作流**：将多步骤任务转化为一致且可审计的工作流。
- **互操作性**：在不同的兼容 skills 的代理产品中复用相同的技能。

## 开放开发

Agent Skills 格式最初由 [Anthropic](https://www.anthropic.com/) 开发，作为开放标准发布，并已被越来越多的代理产品采用。该标准欢迎更广泛生态系统的贡献。
